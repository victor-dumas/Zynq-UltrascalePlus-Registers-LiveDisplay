M_APERTURE_64KB (XPPU) Register

M_APERTURE_64KB (XPPU) Register Description

Register NameM_APERTURE_64KB
Relative Address0x0000000044
Absolute Address 0x00FF980044 (LPD_XPPU)
Width32
TyperoRead-only
Reset Value0x00000100
DescriptionNumber of 64-kB Apertures Implemented for IOP CSRs.

The control and status registers for the peripherals are sectioned into 64-kB blocks. Control of each block can be configured.
