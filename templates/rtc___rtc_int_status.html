RTC_INT_STATUS (RTC) Register

RTC_INT_STATUS (RTC) Register Description

Register NameRTC_INT_STATUS
Relative Address0x0000000020
Absolute Address 0x00FFA60020 (RTC)
Width 2
TypewtcReadable, write a 1 to clear
Reset Value0x00000000
DescriptionInterrupt Status.

This register holds the status of the RTC interrupts. READ: 0: not set. 1: interrupt is set. WRITE: 0: ignored. 1: clear the bit.
