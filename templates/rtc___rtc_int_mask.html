RTC_INT_MASK (RTC) Register

RTC_INT_MASK (RTC) Register Description

Register NameRTC_INT_MASK
Relative Address0x0000000024
Absolute Address 0x00FFA60024 (RTC)
Width 2
TyperoRead-only
Reset Value0x00000003
DescriptionInterrupt Mask.

This read-only register is used to provide the status of the RTC interrupt Mask. The Mask bits can be atomically set or cleared by using RTC_INT_EN and RTC_INT_DIS registers.
