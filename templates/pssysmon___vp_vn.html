VP_VN (PSSYSMON) Register

VP_VN (PSSYSMON) Register Description

Register NameVP_VN
Relative Address0x000000000C
Absolute Address 0x00FFA5080C (AMS_PS_SYSMON)
Width16
TyperoRead-only
Reset Value0x00000000
DescriptionVP_VN voltage measurement.

This register can be used to reset the PS SysMon unit. The PS SysMon does not support the VP_VN analog sensor input.
