SEQ_INPUT_MODE1 (PLSYSMON) Register

SEQ_INPUT_MODE1 (PLSYSMON) Register Description

Register NameSEQ_INPUT_MODE1
Relative Address0x0000000134
Absolute Address 0x00FFA50D34 (AMS_PL_SYSMON)
Width16
TyperwNormal read/write
Reset Value0x00000000
DescriptionSequencer Bipolar Input Mode, Group 1.

The VAux sensor sampling circuit can be programmed to be unipolar (unsigned value) or bipolar (signed value). 0: disabled. 1: enabled. Note: UG580 refers to this register as SEQINMODE1.
