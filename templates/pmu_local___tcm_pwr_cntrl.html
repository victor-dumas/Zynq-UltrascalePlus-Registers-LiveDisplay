TCM_PWR_CNTRL (PMU_LOCAL) Register

TCM_PWR_CNTRL (PMU_LOCAL) Register Description

Register NameTCM_PWR_CNTRL
Relative Address0x00000000D0
Absolute Address 0x00FFD600D0 (PMU_LOCAL)
Width32
TypemixedMixed types. See bit-field details.
Reset Value0x01010101
DescriptionRPU TCM Memory Power Control. Reset by POR only.

Controls the on-chip power Switches: 0: power off. 1: power on. All fields can only be read or written by the PMU processor. This register maintains its contents during a System Reset.
