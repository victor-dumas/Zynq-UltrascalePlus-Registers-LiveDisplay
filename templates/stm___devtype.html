DEVTYPE (STM) Register

DEVTYPE (STM) Register Description

Register NameDEVTYPE
Relative Address0x0000000FCC
Absolute Address 0x00FE9C0FCC (CORESIGHT_SOC_STM)
Width32
TyperoRead-only
Reset Value0x00000063
DescriptionType Classification.

Provides a debugger with information about the component when the part number is not recognized. The debugger can then report this information.
