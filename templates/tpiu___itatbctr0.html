ITATBCTR0 (TPIU) Register

ITATBCTR0 (TPIU) Register Description

Register NameITATBCTR0
Relative Address0x0000000EF8
Absolute Address 0x00FE980EF8 (CORESIGHT_SOC_TPIU)
Width32
TyperoRead-only
Reset Value0x00000000
DescriptionThe Integration Test ATB Control Register 0 captures the values of the atvalids, afreadys, and atbytess inputs to the TPIU. To ensure the integration registers work correctly in a system, the value of atbytess is only valid when atvalids, bit [0], is HIGH.
