SEQ_AVERAGE2 (PLSYSMON) Register

SEQ_AVERAGE2 (PLSYSMON) Register Description

Register NameSEQ_AVERAGE2
Relative Address0x000000011C
Absolute Address 0x00FFA50D1C (AMS_PL_SYSMON)
Width16
TyperwNormal read/write
Reset Value0x00000000
DescriptionSequencer Average Config, Group 2.

Select channels for averaging over multiple readings on a per channel basis. 0: disabled. 1: enabled. The enable and averaging size are selected by Config_Reg0[13:12].
