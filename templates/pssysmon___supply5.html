SUPPLY5 (PSSYSMON) Register

SUPPLY5 (PSSYSMON) Register Description

Register NameSUPPLY5
Relative Address0x0000000038
Absolute Address 0x00FFA50838 (AMS_PS_SYSMON)
Width16
TyperoRead-only
Reset Value0x00000000
DescriptionPS IO Bank 503 voltage measurement (supply5).

IO Bank 503 for PSIO3 voltage node powers IO signals PS boot mode, serial config, JTAG, error output, error status, SRST, POR. The voltage measurement is done with a unipolar sampling circuit to produce an unsigned result with a 0 to 6V range.
