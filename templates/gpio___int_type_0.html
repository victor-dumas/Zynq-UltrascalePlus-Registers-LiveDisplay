INT_TYPE_0 (GPIO) Register

INT_TYPE_0 (GPIO) Register Description

Register NameINT_TYPE_0
Relative Address0x000000021C
Absolute Address 0x00FF0A021C (GPIO)
Width32
TypemixedMixed types. See bit-field details.
Reset Value0x03FFFFFF
DescriptionInterrupt Type (GPIO Bank0, MIO)

This register controls whether the interrupt is edge sensitive or level sensitive. This register controls bank0, which corresponds to MIO[25:0].
