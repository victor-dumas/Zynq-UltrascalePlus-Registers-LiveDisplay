PWR_ACK_ERR_FPD (PMU_LOCAL) Register

PWR_ACK_ERR_FPD (PMU_LOCAL) Register Description

Register NamePWR_ACK_ERR_FPD
Relative Address0x0000000344
Absolute Address 0x00FFD60344 (PMU_LOCAL)
Width32
TyperwNormal read/write
Reset Value0x00000000
DescriptionPRDY Status Error for Power Islands in FPD. Reset only by POR.

Used by the ROM code to log error status of each power chain within the FPD. If the PRDY is not received during the power-up or power-down of a chain, an error is logged in the corresponding bit within this register. This register is only reset by the POR Reset and maintains its state through a System Reset.
