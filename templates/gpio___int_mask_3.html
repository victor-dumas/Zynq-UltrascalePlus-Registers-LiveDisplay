INT_MASK_3 (GPIO) Register

INT_MASK_3 (GPIO) Register Description

Register NameINT_MASK_3
Relative Address0x00000002CC
Absolute Address 0x00FF0A02CC (GPIO)
Width32
TyperoRead-only
Reset Value0xFFFFFFFF
DescriptionInterrupt Mask Status (GPIO Bank3, EMIO Bank0)

This register operates in exactly the same manner as INT_MASK_0, except that it reflects bank3, which corresponds to EMIO[31:0].
