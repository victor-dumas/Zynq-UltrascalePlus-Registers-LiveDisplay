ITATBDATA0 (TPIU) Register

ITATBDATA0 (TPIU) Register Description

Register NameITATBDATA0
Relative Address0x0000000EEC
Absolute Address 0x00FE980EEC (CORESIGHT_SOC_TPIU)
Width32
TyperoRead-only
Reset Value0x00000000
DescriptionThe Integration Test ATB Data Register 0 contains the value of the atdatas inputs to the TPIU. The values are only valid when atvalids is HIGH.
