TPRCR (TPIU) Register

TPRCR (TPIU) Register Description

Register NameTPRCR
Relative Address0x0000000208
Absolute Address 0x00FE980208 (CORESIGHT_SOC_TPIU)
Width32
TyperwNormal read/write
Reset Value0x00000000
DescriptionThis is an eight-bit counter start value that is decremented. A write sets the initial counter value and a read returns the programmed value. On reset this value is set to 0.
