PxCLBU (SATA_AHCI_PORTCNTRL) Register Bit-Field Summary

Field NameBitsTypeReset ValueDescription
CLBU31:0rwNormal read/write0x0Command List Base Address Upper (CLBU):
Indicates the upper 32-bits for the command list base physical address for this port.
This base is used when fetching commands to execute. This register shall be read only '0' for HBAs that do not support 64-bit addressing.
