int_disable (GEM) Register Description

Register Nameint_disable
Relative Address0x000000002C
Absolute Address 0x00FF0B002C (GEM0)
0x00FF0C002C (GEM1)
0x00FF0D002C (GEM2)

Width32
TypemixedMixed types. See bit-field details.
Reset Value0x00000000
DescriptionWriting a 1 to the relevant bit location disables that particular interrupt. This register is write only and when read will return zero.
