PMU_2_IMR (IPI) Register

PMU_2_IMR (IPI) Register Description

Register NamePMU_2_IMR
Relative Address0x0000032014
Absolute Address 0x00FF332014 (IPI)
Width32
TyperoRead-only
Reset Value0x0F0F0301
DescriptionPMU 2 Interrupt Mask (receiver).

Read-only. 0: enabled. 1: masked (disabled). Note: If the ISR bit = 1 (asserted interrupt) and the IMR bit = 0 (not masked), then the IRQ is asserted.
