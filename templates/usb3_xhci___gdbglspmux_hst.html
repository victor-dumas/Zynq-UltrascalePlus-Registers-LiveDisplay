GDBGLSPMUX_HST (USB3_XHCI) Register

GDBGLSPMUX_HST (USB3_XHCI) Register Description

Register NameGDBGLSPMUX_HST
Relative Address0x000000C170
Absolute Address 0x00FE20C170 (USB3_0_XHCI)
0x00FE30C170 (USB3_1_XHCI)
Width32
TypemixedMixed types. See bit-field details.
Reset Value0x003F0000
DescriptionGlobal Debug LSP MUX Register - Host
This register is for internal use only.
If DWC_USB3_PRESERVE_LOGIC_ANALYZER_SELECT is enabled during core configuration, then the default values readout is X (Undefined).
Bit Bash test should not be done on this debug register.
