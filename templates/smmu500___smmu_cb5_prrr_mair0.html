SMMU_CB5_PRRR_MAIR0 (SMMU500) Register

SMMU_CB5_PRRR_MAIR0 (SMMU500) Register Description

Register NameSMMU_CB5_PRRR_MAIR0
Relative Address0x0000015038
Absolute Address 0x00FD815038 (SMMU_GPV)
Width32
TyperwNormal read/write
Reset Value0x00000000
DescriptionPrimary region remap register if AArch32 short descriptor scheme is selected. Controls top-level mapping of the TEX, C, and B memory region attributes. Memory attribute indirection register when AArch32 Long descriptor scheme or AArch64 translation scheme is selected. Provide a revised version of the TEX-Remap system to redirect the selection of memory attributes from the translation table entries.
