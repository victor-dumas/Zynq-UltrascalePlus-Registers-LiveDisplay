ADDR_ERROR (RTC) Register

ADDR_ERROR (RTC) Register Description

Register NameADDR_ERROR
Relative Address0x0000000030
Absolute Address 0x00FFA60030 (RTC)
Width 1
TypewtcReadable, write a 1 to clear
Reset Value0x00000000
DescriptionAddress Decode Error Status.

This register provides the status of the Address Decode Error. READ: 0: not set. 1: interrupt is set. WRITE: 0: ignored. 1: clear the bit.
