EFUSE_CACHE_LOAD (EFUSE) Register Bit-Field Summary

Field NameBitsTypeReset ValueDescription
load 0woWrite-only0x0Reload the eFuse cache from the physical fuses:
1: reload eFuse cache. This bit is self-clearing to 0.
Note: The PMU reads from the eFuse cache, not the physical eFuses. If a cache read error is detected, then reload the eFuse cache.
