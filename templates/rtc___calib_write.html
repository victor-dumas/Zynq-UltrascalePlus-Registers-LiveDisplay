CALIB_WRITE (RTC) Register

CALIB_WRITE (RTC) Register Description

Register NameCALIB_WRITE
Relative Address0x0000000008
Absolute Address 0x00FFA60008 (RTC)
Width21
TypewoWrite-only
Reset Value0x00000000
DescriptionOne Second Time Based.

This register stores the value that is used to generate one second based on the oscillator period
