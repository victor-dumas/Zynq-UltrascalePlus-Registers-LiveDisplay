OEN_2 (GPIO) Register

OEN_2 (GPIO) Register Description

Register NameOEN_2
Relative Address0x0000000288
Absolute Address 0x00FF0A0288 (GPIO)
Width32
TypemixedMixed types. See bit-field details.
Reset Value0x00000000
DescriptionOutput enable (GPIO Bank2, MIO)

This register operates in exactly the same manner as OEN_0, except that it reflects bank2, which corresponds to MIO[77:52].
