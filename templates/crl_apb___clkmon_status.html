CLKMON_STATUS (CRL_APB) Register

CLKMON_STATUS (CRL_APB) Register Description

Register NameCLKMON_STATUS
Relative Address0x0000000140
Absolute Address 0x00FF5E0140 (CRL_APB)
Width16
TypewtcReadable, write a 1 to clear
Reset Value0x00000000
DescriptionClock Monitor Interrupt Status.

This is a sticky register that holds the value of the interrupt until cleared by writing a value of 1 to the bit.
