Program_Register (NAND) Register

Program_Register (NAND) Register Description

Register NameProgram_Register
Relative Address0x0000000010
Absolute Address 0x00FF100010 (NAND)
Width32
TypemixedMixed types. See bit-field details.
Reset Value0x00000000
DescriptionInitiate Controller Operations.

0: no initiation. 1: Initiate a command or operation. Note: Change values in this register only when controller is not communicating with the memory device.
