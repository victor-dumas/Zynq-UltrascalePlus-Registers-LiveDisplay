RTC_INT_EN (RTC) Register

RTC_INT_EN (RTC) Register Description

Register NameRTC_INT_EN
Relative Address0x0000000028
Absolute Address 0x00FFA60028 (RTC)
Width 2
TypewoWrite-only
Reset Value0x00000000
DescriptionInterrupt Enable.

A write to a bit in this register will unmask the corresponding interrupt. 0: ignored. 1: clear the RTC_INT_MASK = 0.
