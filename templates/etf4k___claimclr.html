CLAIMCLR (ETF4K) Register

CLAIMCLR (ETF4K) Register Description

Register NameCLAIMCLR
Relative Address0x0000000FA4
Absolute Address 0x00FE940FA4 (CORESIGHT_SOC_ETF_1)
Width32
TyperwNormal read/write
Reset Value0x00000000
DescriptionThis register is used in conjunction with Claim Tag Set Register, CLAIMSET. This register forms one half of the Claim Tag value. This location enables individual bits to be cleared, write, and returns the current Claim Tag value, read.
