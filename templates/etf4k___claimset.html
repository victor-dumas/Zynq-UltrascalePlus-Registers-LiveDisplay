CLAIMSET (ETF4K) Register

CLAIMSET (ETF4K) Register Description

Register NameCLAIMSET
Relative Address0x0000000FA0
Absolute Address 0x00FE940FA0 (CORESIGHT_SOC_ETF_1)
Width32
TyperwNormal read/write
Reset Value0x0000000F
DescriptionThis is used in conjunction with Claim Tag Clear Register, CLAIMCLR. This register forms one half of the Claim Tag value. This location allows individual bits to be set, write, and returns the number of bits that can be set, read.
