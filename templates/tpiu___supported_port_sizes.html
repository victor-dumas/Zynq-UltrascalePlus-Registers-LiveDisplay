Supported_Port_Sizes (TPIU) Register

Supported_Port_Sizes (TPIU) Register Description

Register NameSupported_Port_Sizes
Relative Address0x0000000000
Absolute Address 0x00FE980000 (CORESIGHT_SOC_TPIU)
Width32
TyperoRead-only
Reset Value0x00000001
DescriptionEach bit location represents a single port size that is supported on the device, that is, 32-1 in bit locations [31:0].
