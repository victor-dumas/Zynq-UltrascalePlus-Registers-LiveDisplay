CTIAPPCLEAR (A53_CTI_2) Register

CTIAPPCLEAR (A53_CTI_2) Register Description

Register NameCTIAPPCLEAR
Relative Address0x0000000018
Absolute Address 0x00FEE20018 (CORESIGHT_A53_CTI_2)
Width32
TypewoWrite-only
Reset Value0x00000000
DescriptionThe CTI Interrupt Acknowledge Register is write-only. A write to this register causes a channel event to be cleared, corresponding to the bit written to.

CTIAPPCLEAR (A53_CTI_2) Register Bit-Field Summary

Field NameBitsTypeReset ValueDescription
APPCLEAR 3:0woWrite-only0x0Clears corresponding bits in the CTIAPPSET register. There is one bit of the register for each channel.When a 1 is written to a bit in this register, the corresponding application trigger is disabled in the CTIAPPSET register. Writing a 0 to any of the bits in this register has no effect.
