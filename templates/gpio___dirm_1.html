DIRM_1 (GPIO) Register

DIRM_1 (GPIO) Register Description

Register NameDIRM_1
Relative Address0x0000000244
Absolute Address 0x00FF0A0244 (GPIO)
Width32
TypemixedMixed types. See bit-field details.
Reset Value0x00000000
DescriptionDirection mode (GPIO Bank1, MIO)

This register operates in exactly the same manner as DIRM_0, except that it reflects bank1, which corresponds to MIO[51:26].
