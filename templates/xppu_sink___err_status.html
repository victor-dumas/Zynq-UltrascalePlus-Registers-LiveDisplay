err_status (XPPU_SINK) Register

err_status (XPPU_SINK) Register Description

Register Nameerr_status
Relative Address0x000000FF00
Absolute Address 0x00FF9CFF00 (LPD_XPPU_SINK)
Width32
TyperwNormal read/write
Reset Value0x00000000
DescriptionAccess Violation Error Status.

One of 2 access violations can occur. Each is defined by an APB address range within the XPPU_Sink address range. 1) Received a poisoned AXI Transaction from XPPU via the APB interface (0xFD4F_0000 to 0xFD4F_FEFF). 2) APB Register Access Error (upper 256 bytes: 0xFD4F_FF00 to 0xFD4F_FFFF). The XPPU_Sink records the first violation it receives. Once a violation is detected, subsequent transactions are not recorded until the XPPU_Sink is reset. The ERR_CTRL [PSLVERR] can enable a violation event to cause the XPPU_Sink to assert the PSLVERR signal back to the APB interconnect.
