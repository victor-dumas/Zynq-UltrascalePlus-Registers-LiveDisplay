INT_EN_5 (GPIO) Register

INT_EN_5 (GPIO) Register Description

Register NameINT_EN_5
Relative Address0x0000000350
Absolute Address 0x00FF0A0350 (GPIO)
Width32
TypewoWrite-only
Reset Value0x00000000
DescriptionInterrupt Enable/Unmask (GPIO Bank5, EMIO Bank2)

This register operates in exactly the same manner as INT_EN_0, except that it reflects bank5, which corresponds to EMIO[95:64].
