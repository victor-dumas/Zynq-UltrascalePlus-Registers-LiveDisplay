VP_VN (PLSYSMON) Register

VP_VN (PLSYSMON) Register Description

Register NameVP_VN
Relative Address0x000000000C
Absolute Address 0x00FFA50C0C (AMS_PL_SYSMON)
Width16
TyperoRead-only
Reset Value0x00000000
DescriptionDifferencial analog input signal Voltage measurment.

Differencial analog input signals VP and VN support an input mode of 1V uni- or bipolar. READ: ADC measurement. WRITE: Any write to this register causes a PL SysMon reset to pulse asserted and de-asserted. The write data has no meaning.
