TCM_RET_CNTRL (PMU_LOCAL) Register

TCM_RET_CNTRL (PMU_LOCAL) Register Description

Register NameTCM_RET_CNTRL
Relative Address0x00000000D4
Absolute Address 0x00FFD600D4 (PMU_LOCAL)
Width32
TypemixedMixed types. See bit-field details.
Reset Value0x00000000
DescriptionRPU TCM Retention Controls.

Retention control signals. 0: active memory. 1: data retention mode. All fields can only be read or written by the PMU processor. This register maintains its contents during a System Reset.
