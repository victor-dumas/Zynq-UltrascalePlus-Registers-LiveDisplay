SUPPLY8 (PSSYSMON) Register

SUPPLY8 (PSSYSMON) Register Description

Register NameSUPPLY8
Relative Address0x0000000204
Absolute Address 0x00FFA50A04 (AMS_PS_SYSMON)
Width16
TyperoRead-only
Reset Value0x00000000
DescriptionVCCO_PSIO2 voltage measurement.

IO Bank 502 for MIO[52:77]. The voltage measurement is done with a unipolar sampling circuit to produce an unsigned result with a 0 to 6V range.
