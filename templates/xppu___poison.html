POISON (XPPU) Register

POISON (XPPU) Register Description

Register NamePOISON
Relative Address0x000000000C
Absolute Address 0x00FF98000C (LPD_XPPU)
Width32
TyperoRead-only
Reset Value0x000FF9C0
DescriptionPoison Address for Sink.

The XPPU supports address poisoning only.

Field NameBitsTypeReset ValueDescription
Reserved31:20roRead-only0x0reserved
BASE19:0roRead-only0xFF9C0Poison Base Address, value fixed at 0x0FF9C0.
The XPPU replaces the incoming AXI address with 0x000_FF9C_0xxx where xxx = bits [11:0] of the incoming address. Downstream, the XPPU_Sink unit responds to the transaction.
