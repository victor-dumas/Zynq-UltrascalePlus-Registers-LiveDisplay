SEQ_INPUT_MODE0 (PLSYSMON) Register

SEQ_INPUT_MODE0 (PLSYSMON) Register Description

Register NameSEQ_INPUT_MODE0
Relative Address0x0000000130
Absolute Address 0x00FFA50D30 (AMS_PL_SYSMON)
Width16
TyperwNormal read/write
Reset Value0x00000000
DescriptionSequencer Bipolar Input Mode, Group 0.

The VP_VN sensor sampling circuit can be programmed to be unipolar (unsigned value) or bipolar (signed value). 0: disabled. 1: enabled. All PL voltage nodes use unipolar sampling circuits except VP_VN and VUser{0:3} support unipolar or bipolar. Note: UG580 refers to this register as SEQINMODE0.
