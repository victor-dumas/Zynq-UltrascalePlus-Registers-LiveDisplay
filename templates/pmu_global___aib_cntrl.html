AIB_CNTRL (PMU_GLOBAL) Register

AIB_CNTRL (PMU_GLOBAL) Register Description

Register NameAIB_CNTRL
Relative Address0x0000000600
Absolute Address 0x00FFD80600 (PMU_GLOBAL)
Width32
TypewoWrite-only
Reset Value0x00000000
DescriptionPS-PL AXI Bus Logic Isolation Requests.

Requests to logically isolate the AXI busses between the PS and PL. 0: no effect. 1: initiate interface shutdown. The PMU will ensure the interface is quiescent before disabling a bus to logically isolate the PS side from the PL side. Write-only. Register is reset only by a POR reset.
