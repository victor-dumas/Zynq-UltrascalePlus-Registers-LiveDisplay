CTIAPPSET (A53_CTI_3) Register

CTIAPPSET (A53_CTI_3) Register Description

Register NameCTIAPPSET
Relative Address0x0000000014
Absolute Address 0x00FEF20014 (CORESIGHT_A53_CTI_3)
Width32
TypewoWrite-only
Reset Value0x00000000
DescriptionThe CTI Application Trigger Set Register is read/write. A write to this register causes a channel event to be raised, corresponding to the bit written to.

CTIAPPSET (A53_CTI_3) Register Bit-Field Summary

Field NameBitsTypeReset ValueDescription
APPSET 3:0woWrite-only0x0Setting a bit HIGH generates a channel event for the selected channel. There is one bit of the register for each channel.Read: 0 = application trigger inactive (reset). 1 = application trigger active.Write: 0 = no effect. 1 = generate channel event.
