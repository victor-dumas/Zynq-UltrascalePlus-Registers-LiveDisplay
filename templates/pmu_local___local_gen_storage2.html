LOCAL_GEN_STORAGE2 (PMU_LOCAL) Register

LOCAL_GEN_STORAGE2 (PMU_LOCAL) Register Description

Register NameLOCAL_GEN_STORAGE2
Relative Address0x0000000308
Absolute Address 0x00FFD60308 (PMU_LOCAL)
Width32
TyperwNormal read/write
Reset Value0x00000000
DescriptionPMU ROM code: ID Value of ROM Undefiend Service Request.

This register can hold the ID value of the last PMU_ROM service that was requested which does not contain a concrete ROM routine. This event is known by the presence of the UNDEFINED_SERVICE error in PMU_LOCAL.PMU_SERV_ERR. (0x400). This register is reserved for the PMU ROM code.
