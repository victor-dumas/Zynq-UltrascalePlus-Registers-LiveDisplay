CTIAPPSET (CTI) Register Bit-Field Summary

Field NameBitsTypeReset ValueDescription
APPSET 3:0woWrite-only0x0Setting a bit HIGH generates a channel event for the selected channel. There is one bit of the register for each channel.Read: 0 = application trigger inactive (reset). 1 = application trigger active.Write: 0 = no effect. 1 = generate channel event.
