RTC_INT_DIS (RTC) Register

RTC_INT_DIS (RTC) Register Description

Register NameRTC_INT_DIS
Relative Address0x000000002C
Absolute Address 0x00FFA6002C (RTC)
Width 2
TypewoWrite-only
Reset Value0x00000000
DescriptionInterrupt Disable.

A write to a bit in this register will mask the corresponding interrupt. 0: ignored. 1: set the RTC_INT_MASK = 1.
