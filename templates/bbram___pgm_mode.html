pgm_mode_value31:0rwNormal read/write0x0Writing 0x757BDF0D to this register will put the BBRAM into programming mode. WARNING, entering programming mode will erase all data in the BBRAM. New data can be loaded into the BBRAM once the PGM_MODE bit is set on the BBRAM_STATUS register.
