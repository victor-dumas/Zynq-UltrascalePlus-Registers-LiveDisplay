CLAIMSET (STM) Register

CLAIMSET (STM) Register Description

Register NameCLAIMSET
Relative Address0x0000000FA0
Absolute Address 0x00FE9C0FA0 (CORESIGHT_SOC_STM)
Width32
TypewoWrite-only
Reset Value0x0000000F
DescriptionClaim Tag Set.

Used in conjunction with Claim Tag Clear Register, STMCLAIMCLR. This register forms one half of the Claim Tag value. This location allows individual bits to be set, write, and returns the number of bits that can be set, read.

Field NameBitsTypeReset ValueDescription
