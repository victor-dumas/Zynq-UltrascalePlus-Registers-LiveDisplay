OEVTEN (USB3_XHCI) Register

OEVTEN (USB3_XHCI) Register Description

Register NameOEVTEN
Relative Address0x000000CC0C
Absolute Address 0x00FE20CC0C (USB3_0_XHCI)
0x00FE30CC0C (USB3_1_XHCI)
Width32
TypemixedMixed types. See bit-field details.
Reset Value0x00000000
DescriptionOTG Events Enable Register
Setting a bit in this register enables the generation of corresponding events in OEVT and assertion of otg_interrupt due to this event. When the enable bit is 1'b0, the event is
not be set in OEVT and otg_interrupt is not asserted due to this event.
