ECCCFG1 (DDRC) Register

ECCCFG1 (DDRC) Register Description

Register NameECCCFG1
Relative Address0x0000000074
Absolute Address 0x00FD070074 (DDRC)
Width32
TyperwNormal read/write
Reset Value0x00000000
DescriptionECC Configuration Register 1

This register is quasi-dynamic group 3. Group 3 registers can only be written when the controller is empty.

ECCCFG1 (DDRC) Register Bit-Field Summary

Field NameBitsTypeReset ValueDescription
data_poison_bit 1rwNormal read/write0x0Selects whether to poison 1 or 2 bits - if 0 -> 2-bit (uncorrectable) data poisoning, if 1 -> 1-bit (correctable) data poisoning, if ECCCFG1.data_poison_en=1
data_poison_en 0rwNormal read/write0x0Enable ECC data poisoning - introduces ECC errors on writes to address specified by the ECCPOISONADDR0/1 registers
