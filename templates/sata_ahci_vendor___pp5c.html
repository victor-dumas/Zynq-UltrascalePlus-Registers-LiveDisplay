PP5C (SATA_AHCI_VENDOR) Register

PP5C (SATA_AHCI_VENDOR) Register Description

Register NamePP5C
Relative Address0x0000000018
Absolute Address 0x00FD0C00B8 (SATA_AHCI_VENDOR)
Width32
TyperwNormal read/write
Reset Value0x3FFC96A4
DescriptionPort Phy Configuration 5.

Controls the configuration of the Phy Control Retry Interval timing for either Port 0 or Port 1. The Port configured is controlled by the value programmed into the Port Config Register.

PP5C (SATA_AHCI_VENDOR) Register Bit-Field Summary

Field NameBitsTypeReset ValueDescription
RCT31:20rwNormal read/write0x3FFRCT: Rate Change Timer, a value based on the 54.2us for which a SATA device will transmit at a fixed rate ALIGNp after OOB has completed, for a fast SERDES it is suggested that this value be 54.2us / 4
RIT19:0rwNormal read/write0xC96A4RIT: Retry Interval Timer. The calculated value divided by two, the lower digit of precision is not needed.
