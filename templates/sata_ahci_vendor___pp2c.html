PP2C (SATA_AHCI_VENDOR) Register

PP2C (SATA_AHCI_VENDOR) Register Description

Register NamePP2C
Relative Address0x000000000C
Absolute Address 0x00FD0C00AC (SATA_AHCI_VENDOR)
Width32
TyperwNormal read/write
Reset Value0x28184D1B
DescriptionPort Phy Configuration 2.

Controls the configuration of the Phy Control OOB timing for the COMINIT parameters for either Port 0 or 1. The Port configured is controlled by the value programmed into the Port Config Register.
