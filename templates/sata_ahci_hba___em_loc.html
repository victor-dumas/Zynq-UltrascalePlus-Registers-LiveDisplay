EM_LOC (SATA_AHCI_HBA) Register

EM_LOC (SATA_AHCI_HBA) Register Description

Register NameEM_LOC
Relative Address0x000000001C
Absolute Address 0x00FD0C001C (SATA_AHCI_HBA)
Width32
TyperoRead-only
Reset Value0x00000000
DescriptionEnclosure Management Location

The enclosure management location register identifies the location and size of the enclosure management message buffer.

EM_LOC (SATA_AHCI_HBA) Register Bit-Field Summary

Field NameBitsTypeReset ValueDescription
OFST31:16roRead-only0x0Offset (OFST): The offset of the message buffer in Dwords from the beginning of the ABAR.
SZ15:0roRead-only0x0Buffer Size (SZ): Specifies the size of the transmit message buffer area in Dwords.
If both transmit and receive buffers are supported, then the transmit buffer begins at ABAR[EM_LOC.OFST*4] and the receive buffer directly follows it.
If both transmit and receive buffers are supported, both buffers are of the size indicated in the Buffer Size field.
A value of '0' is invalid.
