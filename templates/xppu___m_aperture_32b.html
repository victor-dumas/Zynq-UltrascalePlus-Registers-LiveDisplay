M_APERTURE_32B (XPPU) Register

M_APERTURE_32B (XPPU) Register Description

Register NameM_APERTURE_32B
Relative Address0x0000000040
Absolute Address 0x00FF980040 (LPD_XPPU)
Width32
TyperoRead-only
Reset Value0x00000080
DescriptionNumber of 32-Byte Apertures Implemented for IPI.

Of 128 possible apertures, only 8 are actually configured since there are 8 groups of IPI registers in memory space 0xFF30_0000 to 0xFF37_FFFF. Each aperture is aligned to a 4-kB page, but the IPI register sets are aligned to 64-kB page.
