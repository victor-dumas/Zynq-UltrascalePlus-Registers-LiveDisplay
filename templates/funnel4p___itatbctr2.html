ITATBCTR2 (FUNNEL4P) Register

ITATBCTR2 (FUNNEL4P) Register Description

Register NameITATBCTR2
Relative Address0x0000000EF0
Absolute Address 0x00FE920EF0 (CORESIGHT_SOC_FUNN_1)
0x00FE930EF0 (CORESIGHT_SOC_FUNN_2)
Width32
TyperwNormal read/write
Reset Value0x00000000
DescriptionThe Integration Test ATB Control 2 Register performs different functions depending on whether the access is a read or a write:
* A write outputs data on atreadysn, where n is defined by the status of the ATB Funnel Control Register at 0x000
* A read returns the data from atreadym.
