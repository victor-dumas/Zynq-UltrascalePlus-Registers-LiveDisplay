IOU_COHERENT_CTRL (IOU_SLCR) Register

IOU_COHERENT_CTRL (IOU_SLCR) Register Description

Register NameIOU_COHERENT_CTRL
Relative Address0x0000000400
Absolute Address 0x00FF180400 (IOU_SLCR)
Width32
TyperwNormal read/write
Reset Value0x00000000
DescriptionAXI Coherency selection

Select the read/write transaction coherency and bufferability parameters for DMA AXI transactions of misc. IOP controllers. 0: non-cacheable and non-bufferable. 1: cacheable and bufferable.
