ERROR_SIG_EN_1 (PMU_GLOBAL) Register Description

Register NameERROR_SIG_EN_1
Relative Address0x0000000584
Absolute Address 0x00FFD80584 (PMU_GLOBAL)
Width32
TypewoWrite-only
Reset Value0x00000000
DescriptionSystem Errors to PL; Interrupt Enable, Reg 1.

0: no effect. 1: enable interrupt (sets mask = 0). Write-only. For details on the bit fields, refer to the ERROR_STATUS_1 register description. Register is reset only by the PS_POR_B reset signal pin.
