ATTR_94 (PCIE_ATTRIB) Register Bit-Field Summary

Field NameBitsTypeReset ValueDescription
attr_ll_replay_timeout_func 1:0rwNormal read/write0x1Defines how LL_REPLAY_TIMEOUT is to be used, if enabled with LL_REPLAY_TIMEOUT_EN (otherwise, this is not used).
0 = Absolute Value on LL_REPLAY_TIMEOUT
1 = Add to the built-in table value
2 = Subtract from the built-in table value
It is the user's responsibility to ensure that if "1" is chosen, the timeout value does not overflow the 15-bit field. The core will prevent underflows if option "2" is chosen (final value will be 0 for that case).; EP=0x0001; RP=0x0001
