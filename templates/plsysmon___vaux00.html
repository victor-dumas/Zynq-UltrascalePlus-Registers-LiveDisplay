VAUX00 (PLSYSMON) Register

VAUX00 (PLSYSMON) Register Description

Register NameVAUX00
Relative Address0x0000000040
Absolute Address 0x00FFA50C40 (AMS_PL_SYSMON)
Width16
TyperoRead-only
Reset Value0x00000000
DescriptionAuxiliary ch 0 Voltage measurement (VAux0).

These registers are used in coordination with the external analog multiplexor via PL fabric. The analog inputs to the PL SysMon are vauxp[15:0] and vauxn[15:0]. The voltage measurement can be done with a unipolar or bipolar sampling circuit. Unipolar circuits produce an unsigned result with a 0 to 1V range. Bipolar circuits produce a signed result with a +/- 0.5V range.
