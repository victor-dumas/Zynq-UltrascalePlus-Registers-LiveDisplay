PIDR1 (STM) Register

PIDR1 (STM) Register Description

Register NamePIDR1
Relative Address0x0000000FE4
Absolute Address 0x00FE9C0FE4 (CORESIGHT_SOC_STM)
Width32
TyperoRead-only
Reset Value0x000000B9
DescriptionPID - Part Number and Designer Identify.

Part of the set of Peripheral Identification registers. Contains part of the designer specific part number and part of the designer identity.
