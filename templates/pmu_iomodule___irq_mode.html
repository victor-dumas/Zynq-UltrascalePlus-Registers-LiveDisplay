IRQ_MODE (PMU_IOMODULE) Register

IRQ_MODE (PMU_IOMODULE) Register Description

Register NameIRQ_MODE
Relative Address0x000000000C
Absolute Address 0x00FFD4000C (PMU_IOMODULE)
Width32
TypewoWrite-only
Reset Value0x00000000
DescriptionInterrupt Mode Register

This register is used to define which interrupts use fast interrupt mode. All bits set to 1 use fast interrupt mode.
