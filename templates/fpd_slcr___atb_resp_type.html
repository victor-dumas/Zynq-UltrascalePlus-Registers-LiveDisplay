ATB_RESP_TYPE (FPD_SLCR) Register

ATB_RESP_TYPE (FPD_SLCR) Register Description

Register NameATB_RESP_TYPE
Relative Address0x0000006018
Absolute Address 0x00FD616018 (FPD_SLCR)
Width32
TypemixedMixed types. See bit-field details.
Reset Value0x00000007
DescriptionTimed-out AXI Response Selection.

ATB's response to a master on AXI when it sees a timeout (enable using the ATB_RESP_EN register). 0: ATB will send an OKAY. 1: ATB will assert SLVERR. Software must ensure that the corresponding field in the ATB_RESP_EN register is cleared = 0 before changing a bitfield value in this register.
