BRPR (CAN) Register

BRPR (CAN) Register Description

Register NameBRPR
Relative Address0x0000000008
Absolute Address 0x00FF060008 (CAN0)
0x00FF070008 (CAN1)
Width32
TyperwNormal read/write
Reset Value0x00000000
DescriptionBaud Rate Prescaler

The BRPR can be programmed to any value in the range 0-255. The actual value is 1 more than the value written into the register. Please refer to the CAN chapter for more details.
