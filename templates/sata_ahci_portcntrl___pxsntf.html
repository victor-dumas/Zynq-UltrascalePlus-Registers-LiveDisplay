PxSNTF (SATA_AHCI_PORTCNTRL) Register

PxSNTF (SATA_AHCI_PORTCNTRL) Register Description

Register NamePxSNTF
Relative Address0x000000003C
Absolute Address 0x00FD0C013C (SATA_AHCI_PORT0_CNTRL)
0x00FD0C01BC (SATA_AHCI_PORT1_CNTRL)
Width32
TypemixedMixed types. See bit-field details.
Reset Value0x00000000
DescriptionPort x Serial ATA Notification (SCR4: SNotification)

Used to determine if asynchronous notification events have occurred for directly connected devices and devices connected to a Port Multiplier.
