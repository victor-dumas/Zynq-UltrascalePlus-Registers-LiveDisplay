BBCTLR (A53_ETM_1) Register Bit-Field Summary

Field NameBitsTypeReset ValueDescription
MODE 8rwNormal read/write0x0Mode bit:
0: Exclude mode. Branch broadcasting is not enabled in the address range that RANGE defines. If RANGE==0 then branch broadcasting is enabled for the entire memory map.
1: Include mode. Branch broadcasting is enabled in the address range that RANGE defines. If RANGE==0 then the behavior of the trace unit is constrained UNPREDICTABLE. That is, the trace unit might or might not consider any instructions to be in a branch broadcast region.
