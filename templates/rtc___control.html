CONTROL (RTC) Register Bit-Field Summary

Field NameBitsTypeReset ValueDescription
Battery_Disable31woWrite-only0x0Enables the RTC. By writing a 0 to this bit, RTC will be powered off and the only module that potentially draws current from the battery will be BBRAM. The value read through this bit does not necessarily reflect whether RTC is enabled or not. It is expected that RTC is enabled every time it is being configured. If RTC is not used in the design, FSBL will disable it by writing a 0 to this bit.
0: RTC power off.
1: RTC power on.

SLVERR_Enable 0rwNormal read/write0x0Enables SLVERR (Slave Error) if there is an access to an invalid register address. By default, this feature is disabled.
