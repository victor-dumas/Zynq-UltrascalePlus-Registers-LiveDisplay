AXIPC (SATA_AHCI_VENDOR) Register

AXIPC (SATA_AHCI_VENDOR) Register Description

Register NameAXIPC
Relative Address0x0000000024
Absolute Address 0x00FD0C00C4 (SATA_AHCI_VENDOR)
Width32
TypemixedMixed types. See bit-field details.
Reset Value0x00000000
DescriptionAXI PROT Control.

Controls the value of the AWPROT and ARPROT used to distinguish each address operation on the address bus.
