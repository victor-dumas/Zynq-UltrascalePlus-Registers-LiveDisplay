HEER (STM) Register

HEER (STM) Register Description

Register NameHEER
Relative Address0x0000000D00
Absolute Address 0x00FE9C0D00 (CORESIGHT_SOC_STM)
Width32
TyperwNormal read/write
Reset Value0x00000000
DescriptionEnable Hardware Events for Trace

This read/write register is used to enable hardware events to generate trace.The register defined one bit per hardware event. Writing 1 enables the appropriate hardware event, writing 0 disables the appropriate hardware event.
