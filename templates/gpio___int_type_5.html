INT_TYPE_5 (GPIO) Register

INT_TYPE_5 (GPIO) Register Description

Register NameINT_TYPE_5
Relative Address0x000000035C
Absolute Address 0x00FF0A035C (GPIO)
Width32
TyperwNormal read/write
Reset Value0xFFFFFFFF
DescriptionInterrupt Type (GPIO Bank5, EMIO Bank2)

This register operates in exactly the same manner as INT_TYPE_0, except that it reflects bank5, which corresponds to EMIO[95:64].
