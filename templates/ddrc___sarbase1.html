SARBASE1 (DDRC) Register

SARBASE1 (DDRC) Register Description

Register NameSARBASE1
Relative Address0x0000000F0C
Absolute Address 0x00FD070F0C (DDRC)
Width32
TyperwNormal read/write
Reset Value0x00000001
DescriptionSAR Base Address Register 1

This register is static. Static registers can only be written when the controller is in reset.
