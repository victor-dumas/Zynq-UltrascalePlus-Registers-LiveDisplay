PP3C (SATA_AHCI_VENDOR) Register

PP3C (SATA_AHCI_VENDOR) Register Description

Register NamePP3C
Relative Address0x0000000010
Absolute Address 0x00FD0C00B0 (SATA_AHCI_VENDOR)
Width32
TyperwNormal read/write
Reset Value0x0E081906
DescriptionPP3C - Port Phy3 Configuration

Controls the configuration of the Phy Control OOB timing for the COMWAKE parameters for either Port 0 or Port 1. The Port configured is controlled by the value programmed into the Port Config Register.
