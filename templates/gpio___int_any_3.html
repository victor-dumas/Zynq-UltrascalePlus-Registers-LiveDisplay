INT_ANY_3 (GPIO) Register

INT_ANY_3 (GPIO) Register Description

Register NameINT_ANY_3
Relative Address0x00000002E4
Absolute Address 0x00FF0A02E4 (GPIO)
Width32
TyperwNormal read/write
Reset Value0x00000000
DescriptionInterrupt Any Edge Sensitive (GPIO Bank3, EMIO Bank0)

This register operates in exactly the same manner as INT_ANY_0, except that it reflects bank3, which corresponds to EMIO[31:0].
