SERV_LOGCLR_ERR (PMU_LOCAL) Register

SERV_LOGCLR_ERR (PMU_LOCAL) Register Description

Register NameSERV_LOGCLR_ERR
Relative Address0x0000000348
Absolute Address 0x00FFD60348 (PMU_LOCAL)
Width32
TyperwNormal read/write
Reset Value0x00000000
DescriptionLogic Clear Services Log Error Status. Reset only by POR.

Used by the ROM code to log error status for the Logic Clear Services. This register is only reset by the Power-on Reset and maintains its state through a System Reset.
