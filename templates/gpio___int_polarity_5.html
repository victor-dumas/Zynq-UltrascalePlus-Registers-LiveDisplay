INT_POLARITY_5 (GPIO) Register

INT_POLARITY_5 (GPIO) Register Description

Register NameINT_POLARITY_5
Relative Address0x0000000360
Absolute Address 0x00FF0A0360 (GPIO)
Width32
TyperwNormal read/write
Reset Value0x00000000
DescriptionInterrupt Polarity (GPIO Bank5, EMIO Bank2)

This register operates in exactly the same manner as INT_POLARITY_0, except that it reflects bank5, which corresponds to EMIO[95:64].
