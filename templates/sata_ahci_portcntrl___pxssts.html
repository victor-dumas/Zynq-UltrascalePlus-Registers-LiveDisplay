PxSSTS (SATA_AHCI_PORTCNTRL) Register

PxSSTS (SATA_AHCI_PORTCNTRL) Register Description

Register NamePxSSTS
Relative Address0x0000000028
Absolute Address 0x00FD0C0128 (SATA_AHCI_PORT0_CNTRL)
0x00FD0C01A8 (SATA_AHCI_PORT1_CNTRL)
Width32
TyperoRead-only
Reset Value0x00000000
DescriptionPort x serial ATA Status (SCR0: Sstatus) (PxSSTS)

This 32-bit register conveys the current state of the interface and host. The HBA updates it continuously and asynchronously. When the HBA transmits a COMRESET to the device, this register is updated to its reset values.
