PxFB (SATA_AHCI_PORTCNTRL) Register Bit-Field Summary

Field NameBitsTypeReset ValueDescription
FB31:8rwNormal read/write0x0FIS Base Address (FB):
Indicates the 32-bit base physical address for received FISes.
The structure pointed to by this address range is 256 bytes in length.
This address must be 256-byte aligned as indicated by bits 07:00 being read only.
When FIS-based switching is in use, this structure is 4KB in length and the address shall be 4KB aligned (refer to section 9.3.3).
