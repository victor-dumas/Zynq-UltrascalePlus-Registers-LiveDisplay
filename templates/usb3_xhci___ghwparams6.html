This bit indicates whether the parameter `DWC_USB3_EN_OTG is set to 2. In other words, it indicates that whether the controller supports OTG 3.0.

The application uses this bit to determine the controller's RSP/HNP support.
If DWC_USB3_EN_OTG=2,
- 1'b0: RSP and HNP support is not enabled. The only exception for this rule is for SSPC-OTG devices where RSP support is not enabled, but HNP support is enabled. (Refer to the
OCFG.SSPC-OTG bit.)
- 1'b1: RSP and HNP support is enabled
If DWC_USB3_EN_OTG=1,
- 1'b0: HNP support is not enabled;
- 1'b1: HNP support is enabled;
