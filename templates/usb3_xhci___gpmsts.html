GPMSTS (USB3_XHCI) Register

GPMSTS (USB3_XHCI) Register Description

Register NameGPMSTS
Relative Address0x000000C114
Absolute Address 0x00FE20C114 (USB3_0_XHCI)
0x00FE30C114 (USB3_1_XHCI)
Width32
TypemixedMixed types. See bit-field details.
Reset Value0x00000000
DescriptionGlobal Power Management Status Register
This debug register gives information on which event caused the hibernation exit. It provides internal status and state machine information, and is for
use only for debugging purposes.
This register is not applicable in USB 2.0-only mode.
